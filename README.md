# Vishesh Panchal Portfolio

This is a personal portfolio website for **Vishesh Panchal**. It showcases skills, projects, and contact information with modern design and smooth animations.

## Features

- Responsive design for all devices
- Animated sections using GSAP and ScrollTrigger
- Projects showcase with links
- Contact form
- Social media links

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- [GSAP](https://greensock.com/gsap/) for animations
- [Font Awesome](https://fontawesome.com/) for icons

## Getting Started

1. **Clone or Download** this repository to your local machine.
2. Open `index.html` in your web browser.

No build tools or server required.

## Folder Structure

```
portfolio/
├── index.html
├── README.md
└── (other assets if added)
```

## Customization

- Update your name, about, skills, and project details in `index.html`.
- Add more projects or skills as needed.
- Update social media links in the footer.
